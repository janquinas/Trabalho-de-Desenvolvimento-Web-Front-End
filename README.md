Verde Viva — Jardinagem e Paisagismo

Este repositório contém o projeto desenvolvido para a disciplina de Front-End, com o objetivo de criar um site funcional para apresentar e promover os serviços de jardinagem e paisagismo oferecidos pela empresa fictícia Verde Viva. O trabalho foi realizado por um grupo de cinco integrantes, com foco em organização, clareza visual e aplicação prática dos conceitos estudados.

Descrição do Produto/Serviço:

Um serviço especializado em jardinagem e paisagismo, atendendo residências, condomínios e ambientes corporativos. A plataforma apresenta:
Serviços detalhados (manutenção, poda, projetos de paisagismo etc.);
Galeria de trabalhos;
Informações sobre a empresa;
Fluxo de contratação com simulação de checkout.
O objetivo é criar uma experiência simples e intuitiva, aproximando o usuário do serviço e exibindo a qualidade e identidade visual da marca.

----------------------------------------------

Tecnologias Utilizadas:

HTML5 — Estruturação das páginas

CSS3 — Estilização, responsividade e identidade visual

JavaScript — Funcionalidades e interações

Bootstrap — Componentes, layout responsivo e grid system

-----------------------------------------------

 Instruções para Execução Local

Clone o repositório localmente com o comando:

git clone https://github.com/janquinas/Trabalho-de-Desenvolvimento-Web-Front-End.git


Entre na pasta do projeto:

cd Trabalho-de-Desenvolvimento-Web-Front-End

Abra o arquivo index.html no navegador — por exemplo, dando dois cliques diretamente ou abrindo pelo seu editor de código/IDE.

(Opcional) Se quiser facilitar o desenvolvimento e visualizar atualizações em tempo real, use uma extensão ou ferramenta como Live Server no VS Code ou similar.

O site deve carregar normalmente — como é um projeto estático (HTML, CSS, JS e Bootstrap), não há necessidade de instalar dependências adicionais nem rodar servidor back-end.
